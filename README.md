# CustomEditorWindowDemo
自定义EditorWindow示例 - by tigerWng

* ver 0.0.1
  
  使用继承ScriptableObject的派生类作为序列化数据类，在自定义editor window中显示数据。
    * 类中只添加了一个容器类list保存结构体subtitle data类型。
    * 在window面板显示的部分由序列化类和结构体的OnGUI函数完成。
